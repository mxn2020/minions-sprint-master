---
title: Installation
description: Install Minions Sprint-master packages
---

## TypeScript / Node.js

```bash
npm install @minions-sprint-master/sdk minions-sdk
```

## Python

```bash
pip install minions-sprint-master
```

## CLI

```bash
npm install -g @minions-sprint-master/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
