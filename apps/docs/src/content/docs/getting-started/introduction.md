# Minions Sprint-master

Agent to organize and sort daily granular task priority

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-sprint-master/sdk`) — Core library for Node.js
- **Python SDK** (`minions-sprint-master`) — Python bindings
- **CLI Tool** (`@minions-sprint-master/cli`) — Command-line interface
- **Web App** — Interactive playground at [sprint-master.minions.wtf](https://sprint-master.minions.wtf)
